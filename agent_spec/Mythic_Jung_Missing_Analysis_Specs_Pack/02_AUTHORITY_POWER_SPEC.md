# 02 — Authority & Power Relationship Spec

## Purpose
Model the user's **relationship to imposed power, hierarchy, and authority**.

This is NOT leadership style.
It is power tolerance and response.

## Where it appears
- Functioning tab (new subsection): **Power Stance**
- Life Domains: Leadership & Authority

## Core dimensions
- Authority acceptance:
  - default trust
  - conditional trust
  - default skepticism
- Power response:
  - comply
  - work around
  - infiltrate
  - openly defy
- Competence threshold:
  - high tolerance
  - medium
  - low (incompetence triggers disengagement)

## Generation rules
- Derive primarily from characters.
- Reinforce via Ego Position + Shadow Proximity.
- Avoid moral judgment.
- Describe pattern as *structural necessity*.

## Output requirements
- Concrete examples:
  - workplace
  - institutions
  - family systems
- Warning section:
  “When misaligned, this shows up as ___.”

## Evidence
- Reference Ego + Shadow questions.
